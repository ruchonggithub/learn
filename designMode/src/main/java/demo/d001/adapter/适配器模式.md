# 适配器模式

适配器模式有以下特点  
(1)接收的接口已知且固定,如:手机上的充电口  
(2)提供的接口未知,如:美国提供110v交流电,中国提供220v交流电,火星提供+5v直流电  
(3)适配前和适配后,他们的本质没有变化,如:适配前后,他们都是电压

所以,当拥有多个输入源且输出目标固定的时候,可以使用适配器模式
